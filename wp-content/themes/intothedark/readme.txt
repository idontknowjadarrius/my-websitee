== Description ==

Into The Dark is an elegant and flexible WordPress theme designed for a sleek dark style with captivating animations. This theme offers a custom template with a transparent menu, customizable button in the transparent menu template, and a three-column footer widget area.

## Features

- Sleek dark style
- Captivating animations
- Custom template with transparent menu
. Custom blank template (without title) - entirely customizable with Gutenberg blocks
- Customizable button in the transparent menu template in customize -> Button Settings (button text and icon, check "Arrow")
- Scroll to Top in in customize -> Button Scroll to Top (check "Show Scroll to Top") 
- Three-column footer widget area

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Into The Dark in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to your taste.

== Privacy ==
Into The Dark theme don't uses LocalStorage to save the setting.
No data is saved in the database or transferred.
